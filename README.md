# ModernMCA: Multiple Correspondence Analysis for scikit-learn

Multiple Correspondence Analysis (MCA) is a multivariate statistical technique originally proposed by Herman Otto Hartley and later popularized by the French statistician Jean-Paul Benzécri. 

Standard PCA applied to categorical data is mathematically flawed. MCA solves this by acting as the categorical counterpart to PCA. Instead of maximizing variance based on Euclidean distances of continuous variables, MCA uncovers latent structures by analyzing the **Chi-Square distances** between categories within an indicator matrix.

### Objectives

The primary objective of applying MCA is to find, quantify, and visualize underlying structures, associations, and patterns within complex categorical datasets. It achieves this by projecting both the individuals (rows) and the categories (columns) into a shared, lower-dimensional geometric space (typically 2D or 3D).

By operating on the Chi-Square metric, the distance between two individuals in the projected space reflects the similarity of their profiles (the categories they chose). Similarly, the distance between two categories reflects their tendency to be chosen by the same individuals. The result is a set of orthogonal vectors (factors) that capture the maximum possible categorical inertia (variance).

For Bourdieu, the spatial distances generated by MCA are direct representations of social distances, mapping how economic, cultural, and social capitals are distributed and how they shape habitus and lifestyle.

### Technical Overview

Unlike naive implementations, `ModernMCA` is mathematically rigorous and accounts for the structural artifacts of indicator matrices.

**1. The Indicator Matrix and Chi-Square Metric:**
The algorithm begins by transforming the categorical dataset $X$ of dimensions $N \times Q$ (where $N$ is the number of individuals and $Q$ is the number of variables) into a complete disjunctive table, or Indicator Matrix $Z$, of dimensions $N \times J$ (where $J$ is the total number of categories). 
The probability matrix $P = Z / N$ is computed, alongside the row masses ($r$) and column masses ($c$). The standardized residual matrix $S$ is then constructed using the Chi-Square metric:
$$S = \frac{P - r c^T}{\sqrt{r c^T}}$$

**2. Singular Value Decomposition (SVD):**
We compute the SVD of the standardized matrix: $S = U \Sigma V^T$. The raw eigenvalues $\lambda_{raw}$ (principal inertias) are derived from the singular values ($\Sigma^2$). These raw values are strictly used to compute the valid coordinates for rows and columns in the spatial projection.

**3. The Dimensionality Artifact (Benzécri & Greenacre Corrections):**
A known geometric artifact of MCA is that the raw eigenvalues severely underestimate the true explained variance. This happens because the implicit Burt Matrix ($Z^T Z$) contains diagonal blocks of identity matrices, artificially inflating the total inertia. 
`ModernMCA` solves this natively through two correction strategies:
* **Benzécri Correction (`correction='benzecri'`):** Recalculates the eigenvalues to only account for components whose inertia is greater than the average ($1/Q$). It provides a highly optimistic, yet sociologically relevant, view of the explained variance.

* **Greenacre Correction (`correction='greenacre'` - Default):** Provides a more rigorous mathematical bound by calculating the exact off-diagonal inertia of the Burt matrix, avoiding the inflation seen in Benzécri's approach while accurately reflecting the true explanatory power of the axes.

### How to Use

`ModernMCA` is designed to be a fully compliant `scikit-learn` transformer. It easily integrates into complex data pipelines, allowing for seamless transitions between data engineering, statistical modeling, and interactive visualization.

#### 1. Basic Fitting and Transformation

```python
import pandas as pd
from sklearn.pipeline import Pipeline
from your_library import ModernMCA

# Assuming df is a pandas DataFrame with strictly categorical/object columns
df = pd.read_csv('consultancies_data.csv')

# Initialize the MCA model with Greenacre's rigorous correction
mca = ModernMCA(n_components=5, correction='greenacre')

# Fit and transform the data 
# Output: A DataFrame with the coordinates of the individuals in the new space
individuals_coords = mca.fit_transform(df)

print(f"Total Inertia Explained: {mca.explained_variance_ratio_[:2].sum() * 100:.2f}%")
```



#### 2. Extracting Deep Statistics

To conduct a true sociological or structural analysis, you need the coordinates and the absolute contributions of the categories to each axis. The contribution tells you which variables are mathematically pulling the dimensions.


```Python
# Get row coordinates (Individuals/Organizations)
row_stats = mca.get_row_stats()

# Get column coordinates (Categories) and their Contributions
col_coords, col_contrib = mca.get_column_stats()

# Check the categories that contribute the most to Axis 1
top_contributors_dim1 = col_contrib.sort_values(by='Contr 1', ascending=False).head(10)
print(top_contributors_dim1)
```
